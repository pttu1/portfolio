  • Timing channel Security for NoC Router

    This project was implemented to secure the vulnerabilities that attackers exploit in On-chip Networks routers. VHDL was used to build build and modify both packet switching and circuit switching routers by partitioning the routings on the network to security levels, and using methodologies like TDM, granted request based on security levels to allow certain security level packets to be propagated in the network. Y-X routing algorithm was used to route the packets from a source node(router) to the destination node

    VHDL, FPGA
